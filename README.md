## 公開手順

１．このZIPを解凍します。  
２．中身をすべてリポジトリの直下へアップロードします。  
３．設定画面で公開機能を有効にします。  
４．公開元は「main」「root」を選択します。  
５．数分後に公開URLが表示されます。

トップページは `index.html` です。
